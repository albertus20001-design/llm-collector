# LLM Info

一个基于 Mintlify 的模型 / 厂商 / 平台情报文档站。

## 收录内容
- 新模型发布
- 下线、弃用与停服
- 价格变化
- API 与兼容性更新
- 聚合平台 / 路由平台变化

## 使用方式
1. 先查看 `guide/sources`，并在 [`sources/index.yml`](https://github.com/albertus20001-design/llm-collector/blob/main/sources/index.yml) 中新增来源
2. 在 `guide/schedule` 中记录采集频率
3. 在 `guide/output` 中整理并发布结论
