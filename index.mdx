# Intel Collector

A Mintlify docs site for tracking model/vendor/platform intelligence.

## What it collects
- New model launches
- Deprecations and shutdowns
- Pricing changes
- API and compatibility updates
- Aggregator/platform changes

## How to use
1. Add a source under `collection/sources`
2. Record collection cadence in `collection/schedule`
3. Publish findings in `collection/output`
